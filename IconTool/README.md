# WebTools - 纯前端在线工具集

这是一个纯前端开发的在线工具集合，所有处理均在浏览器本地完成，不上传服务器，确保数据隐私安全。

## 工具列表

### 1. [IconTool](./IconTool/) - 图片取色与颜色替换
支持上传图片、选取特定颜色并进行替换。提供容忍度调节和局部区域限定功能。

### 2. [BannerTool](./BannerTool/) - 网页横幅制作 (开发中)
用于快速生成网页 Banner、社交媒体封面等。
